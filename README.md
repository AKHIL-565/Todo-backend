 Todo App Backend  
Node.js | Express | PostgreSQL (Supabase)

This repository contains the backend API for the Todo List Application.  
It provides all CRUD functionalities — create, read, update, and delete tasks — and connects to a live PostgreSQL database hosted on Supabase.

This backend is part of a full-stack assignment for the Internship Task.  




 Features
- Create a new task  
- Get all tasks  
- Edit a task (update text)  
- Mark task as completed / uncompleted  
- Delete a task  
- Clean folder structure (controllers, routes, services)
